口语
RS: 50%以上有效内容，语速平稳，发音清晰，句子流畅，几乎每句话都是3-4个意群词组构成的，要听结构和词组
DI、RL不需要关注逻辑
RL、DI、SGD不需要在意语法，注重内容，也不用过于追究内容，流利度最重要+发音清晰
RL、SGD机经命中率一般，但供分率高，关注口语表达需要流畅的说一篇文章
RTS(respond to a situation)接近于文本版的DI
RTS情景描述：岁月静好->突发情况->你如何解决

听力
人工审核：SST 
羊驼机经命中率最高的几道题：
WE、SWT、SST、WFD（看高频就可以），考前必刷高频
除RA、RS能刷全题库，尽量刷其他至少要刷羊驼本周高频
WFD顺序不重要、写错不扣分（不确定的单词可试词）
复习的时候要按照意群去复习
WFD如果愿意可以记（记录所有主要实词的首字母）
WFD必须首字母大写，句尾加句号
FIBL机经命中率低，
HIW送分题，选错倒扣分，几乎没有机经
所有写的题目都要加句号，如果不加会被当成0分处理
SST和WE需要关注拼写，错一个扣一分，WE如果错2个则拼写分全扣
SWT不考拼写，但是关键词写错会被扣内容分

写作
写作部分SWT加入了人工审核（内容板块）
SWT、WE都是独立计时SWT有2题每题10分钟，WE有1题20分钟
SWT 中content或form为0分，整题为0分，先通过机器审核再进入人工审核
WE也加入了人工审核

阅读
阅读部分单词积累可以通过PTE背单词APP或者WFD、SST来积累单词
网上找FIB固定搭配和语法文档
语法：情态动词后面只能接动词原形
一般一个一群的结束都是以名词结束
RO一般全称的句子靠前，缩写靠后，整段有多个概念的话同一个概念会在一起，会抽象到具体，一般是总分总写法
